# DESCRIPTION

Activity : Creation of our own library of functions and its makefile.

Goal : Discover and anchor the key knowledge and skills of the programming's fundamentals. Understanding the function's codes and the C language to its core. Experiencing problem solving, peer-learning and thoughtful ways of using AI.

Overview : LIBFT is a repertoire of tools. It's now a reference that we can go back to in order to reuse the functions we coded.


# INSTRUCTIONS

To make sure that the library is complete ; please check that every prototype's function appears in the header file.

Compilation process : 
- Use the command "make" to create the *.o files and the executable file.
- Use "clean" to delete the *.o files.
- Use "fclean" to delete the *.o files and the executable file.
- Use "re" to delete all the files and rerun the process.
